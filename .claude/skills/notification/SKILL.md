# Notification Skill

## Instructions

1. **Notification triggers**
   - Detect due dates or events
   - Schedule reminders appropriately

2. **Delivery**
   - Use browser notifications where supported
   - Provide graceful fallbacks

3. **User control**
   - Allow opt-in and opt-out
   - Respect user preferences

## Best Practices
- Avoid notification spam
- Deliver timely and relevant alerts
- Fail silently when unsupported
