# Internationalization Skill

## Instructions

1. **Language handling**
   - Externalize user-facing text
   - Support runtime language switching

2. **Localization**
   - Handle date and time formats
   - Support right-to-left languages

3. **Chatbot readiness**
   - Accept multilingual input
   - Normalize language internally

## Best Practices
- Do not hardcode strings
- Default to English with fallbacks
- Validate translations for clarity
